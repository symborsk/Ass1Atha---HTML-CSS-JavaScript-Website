CSS

CSS is a styling language that describes the way a the presentation of a markup language should be. This is most often used to format and style webpages like the one you are on right now.

JavaScript

JavaScript is a high level programming language that is used to make a webpage a little bit smarter. By this I mean that it can react to events making the overall experience for the user a lot nicer.

Html

Html is the language that is used to actually render a webpage. This language uses different tags to be able to interpret content and render it. It decides the structure of your webpage.

Ajax

Ajax is a programming technique used to make a user experience on a website better. It allows us to decouple the data exchange that happens between a client and a server.

XML

XML is a storage files that has a very easy to understand and readable structure. One of the most common uses of this data structure is to tranfer data over the internet.
